# Atelier assets

## `modern-interiors/`

Sprite atlas and characters from [Modern Interiors by LimeZu](https://limezu.itch.io/moderninteriors)
— **full version** (commercial use permitted; redistribution forbidden; credits
required to limezu.itch.io).

See `modern-interiors/LICENSE.txt`. The PixiJS office view (`web/app/office/`)
loads these sprites directly.

Bundled subset (the office view does not need the entire pack):

```
modern-interiors/
├── interiors/
│   ├── 16x16/Interiors_16x16.png, Room_Builder_16x16.png
│   ├── 32x32/Interiors_32x32.png, Room_Builder_32x32.png
│   └── 48x48/Interiors_48x48.png, Room_Builder_48x48.png
├── characters/        # 9 idle sprites (one per dept lead) + Adam variants
├── LICENSE.txt        # Full version conditions
├── README.txt
└── overview.png       # Marketing sheet from upstream
```
