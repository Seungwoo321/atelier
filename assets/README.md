# Atelier assets

## `modern-interiors/`

Sprite atlas and characters from [Modern Interiors by LimeZu](https://limezu.itch.io/moderninteriors)
— **free version**.

License is non-commercial use only. See `modern-interiors/LICENSE.txt`. The
PixiJS office view (`web/app/office/`) loads these sprites directly.

Source files preserved as shipped:

```
modern-interiors/
├── interiors/
│   ├── 16x16/
│   ├── 32x32/
│   └── 48x48/
├── characters/        # Adam, Alex, etc. + idle/run/sit animations
├── LICENSE.txt        # CAN/CAN'T conditions
├── README.txt
└── overview.png       # Marketing sheet of the free contents
```
